Movie Explorer Web App

A React-based movie discovery web application built using Vite and TMDB API.  
Users can search for movies, explore details, and manage their favorite movies with a clean and responsive UI.

Features
- Search movies in real-time
- Add / Remove movies from Favorites
- View movie posters and release dates
- Dynamic routing using React Router
  
Tech Stack
- React.js (Functional Components & Hooks)
- JavaScript
- TMDB REST API
- CSS
